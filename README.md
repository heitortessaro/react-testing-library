# O Projeto

Para esse projeto, a aplicação front-end foi fornecida pela Trybe. Meu objetivo era implementar os testes utilizando Jest e a biblioteca React Testing Library e a biblioteca React Testing Library.

## Habilidades

Neste projeto, foram utilizadas as seguintes abilidades:
* Utilizar os seletores (queries) da React-Testing-Library em testes automatizados.
* Simular eventos com a React-Testing-Library em testes automatizados.
* Testar fluxos lógicos assíncronos com a React-Testing-Library.
* Escrever testes que permitam a refatoração da estrutura dos componentes da aplicação sem necessidade de serem alterados.
* Criar mocks de APIs utilizando fetch.
* Testar inputs.

## Testes desenvolvidos

Os testes desenvolvidos encontram-se no diretório ./src/tests. Cada um dos componentes da aplicação (About, FavoritePokemons, PokemonDetails, ...) foi testado individualmente. Para os testes, a API utilizada foi mockada.

Após baixar o projeto e instalar as dependência, você pode executar os testes implementados utilizando o comando *npm run test*. Já se preferir verificar a cobertura dos testes, rode o comando *npm run test-coverage*.
